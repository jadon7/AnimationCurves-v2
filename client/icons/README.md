# Icons Folder

This folder should contain the following icon files (23x23px PNG):

- `icon-normal.png` - Normal state icon
- `icon-rollover.png` - Hover state icon
- `icon-disabled.png` - Disabled state icon
- `icon-dark-normal.png` - Dark theme normal icon
- `icon-dark-rollover.png` - Dark theme hover icon

For development, you can use placeholder icons or the extension will work without them (just won't show an icon in the menu).

To create icons:
1. Design a 23x23px icon representing animation curves
2. Export in PNG format with transparency
3. Create both light and dark versions
4. Name them according to the list above
